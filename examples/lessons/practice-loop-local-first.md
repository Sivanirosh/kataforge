title: Practice Locally First
---

KataForge is a local practice loop: your code runs in the browser, and drafts, scores, Cursus progress, and checkpoint attempts stay in local storage.

This demo uses only the committed public example katas already in the app: **Two Sum** and **FizzBuzz**.

## The loop

1. Read the prompt and starter code.
2. Edit a small solution.
3. Use **Run Samples** for quick visible feedback while you work.
4. Use **Submit** when you want the full kata result, including hidden tests.
5. Review the result, keep the draft if you need another pass, then continue.

After Two Sum, a lightweight self-check asks whether the local feedback made the next step clear before you move on.
