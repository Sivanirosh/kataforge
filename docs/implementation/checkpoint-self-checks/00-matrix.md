# Checkpoint Self-checks Implementation Matrix

## Status State Machine

- `planned`: in the matrix, workpackage drafted, not started.
- `ready`: workpackage reviewed, open questions resolved, dependencies met.
- `in_progress`: active implementation.
- `blocked`: cannot proceed; blocker is named explicitly.
- `done`: every Definition of Done item is verified.
- `explicitly_deferred`: decision recorded to not implement; rationale and revisit condition are in the workpackage.
